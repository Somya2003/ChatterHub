# ChatterHub – AI Driven Chat Application

## Overview

ChatterHub is an interactive AI-powered chat application that enables real-time text and voice-based communication. Built using Generative AI and modern NLP frameworks, it delivers a ChatGPT-like conversational experience with smooth and natural responses.

## Features

💬 Real-time AI chat interface (text + voice)

🎙️ Speech-to-text conversion for voice input

🤖 ChatGPT-style conversational flow using LLMs

⚡ Fast and interactive UI using Streamlit

🧠 Context-aware responses powered by LangChain

🔊 Seamless voice interaction for better accessibility

## Tech Stack

### Frontend

* Streamlit

### Backend

* Python

### AI / NLP

* Generative AI (LLMs)
* LangChain
* ChatOllama

### Speech Processing

* Speech Recognition API

## Workflow

1. User sends input via text or voice.
2. Voice input is converted into text using Speech Recognition API.
3. Input is processed through LangChain pipeline.
4. ChatOllama generates contextual AI responses.
5. Response is displayed in real time on Streamlit UI.

## Outcome

* Delivered a smooth ChatGPT-like experience.
* Enabled voice + text hybrid interaction for better usability.
* Improved conversational fluency using LLM-based architecture.
